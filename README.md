### SPO-Ratifier

A centralised way to verify a students involvments.

### Tech stack - 
NEXTjs
GO
PostgreSQL

### Running the Project
## Backend
go run main.go

# Backend runs on:

http://localhost:8080

## Frontend
npm install
npm run dev

# Frontend runs on:

http://localhost:3000

### Demo Credentials

Master Admin needs to be seeded into the databse manually. Domain Admin, and Student accounts can be configured through the database as well.
